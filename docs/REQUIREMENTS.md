# trace — 需求文档 / Requirements

> 个人跑步/运动记录 App（类 Keep，去社交/账号）。支持户外跑、室内跑、骑行、步行；
> iCloud 多端同步，深度集成 Apple 健康。最低 iOS 18.0（iPhone）。
> 本文为可追溯需求清单，分期计划见 [ROADMAP.md](./ROADMAP.md)，工程约定见 [CLAUDE.md](../CLAUDE.md)。
